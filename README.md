# webpack demo
node.js 版本: v12.18.3

cd 到新建的空目录 webpackDemo 下, 依次执行如下命令:
```
> nmp init
> npm i webpack webpack-cli -g
> npm i webpack webpack-cli -D
```

```
开发环境:
   运行项目指令:
     webpack 会将打包结果输出到目标路径
     npx webpack-dev-server 只会在内存中打包编译, 没有输出
 ```
